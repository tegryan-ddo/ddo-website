- Have budget but not time

- "Cloud costs are out of control"

**Value Proposition:**

**Pricing Tier:**

## Design Specifications

### Color System

```css
/* Primary - Professional, technical */
--slate-950: #020617;  /* Background (dark) */
--slate-900: #0f172a;  /* Cards, surfaces */
--slate-800: #1e293b;  /* Borders, dividers */

/* Accent - AI, innovation */
--blue-500: #3b82f6;   /* Primary accent */
--cyan-500: #06b6d4;   /* AI/automation elements */
--cyan-400: #22d3ee;   /* Highlights */

/* Status colors */
--emerald-500: #10b981; /* Success, active */
--amber-500: #f59e0b;   /* Warning, attention */
--red-500: #ef4444;     /* Error, critical */

/* Text */
--white: #ffffff;       /* Primary text (dark mode) */
--slate-300: #cbd5e1;   /* Secondary text */
--slate-400: #94a3b8;   /* Muted text */
```

### Typography

```css
/* Headings */
font-family: 'Inter', system-ui, sans-serif;
h1: 4rem (64px), font-weight: 700
h2: 2.5rem (40px), font-weight: 600
h3: 1.5rem (24px), font-weight: 600

/* Body */
font-family: 'Inter', system-ui, sans-serif;
body: 1rem (16px), font-weight: 400
small: 0.875rem (14px)

/* Code/Technical */
font-family: 'JetBrains Mono', monospace;
code: 0.875rem (14px)
```

### Component Patterns

**Cards**: Glass effect with subtle borders
```css
background: rgba(15, 23, 42, 0.8);
backdrop-filter: blur(12px);
border: 1px solid rgba(255, 255, 255, 0.1);
border-radius: 12px;
```

**Buttons**: Bold, clear hierarchy
```css
/* Primary */
background: linear-gradient(to right, #3b82f6, #06b6d4);
/* Secondary */

```
